# JD Logistics 京东快递助手

帮你搞定京东快递的查件、选服务、估运费和寄件准备——覆盖京东自营物流的日常使用场景。

## What it does
- 快递单号追踪和时效预估
- 京尊达/特快送/特惠送等服务类型对比
- 城际运输时效和费用估算
- 退货上门取件流程指导

## Example scenarios

**1. 京东快递查件**
> 👤 京东快递单号 JDK123456789 到哪里了？
> 🤖 当前状态：快件已到达北京通州分拣中心，预计今日达。如果走的是京东次日达服务，SLA 状态为正常。点击订单详情可查看派件小哥联系方式。

**2. 三种服务怎么选**
> 👤 京东快递的京尊达、特快送和特惠送有什么区别？
> 🤖 简单说——京尊达：专人专车，适合贵重/送礼，单价最高；特快送：主力快递，1-3 天，大多数电商订单默认这个；特惠送：经济型，3-5 天，适合不着急的包裹。对时效敏感选特快送，对价格敏感选特惠送，送贵重物品且需要仪式感选京尊达。

**3. 退货上门取件忘了取件码**
> 👤 在京东买了个手机壳要退货，约了明天上门但忘了取件码放哪了，怎么办？
> 🤖 别急，取件码可以在京东 App 的"我的订单 → 退换/售后"里重新查看，或者在京东快递小程序中找到上门取件编号。如果还是找不到，联系京东客服可重新发送。记得把包裹准备好——原包装 + 退换货单，等快递员按预约时间上门就行。
